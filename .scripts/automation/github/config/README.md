# Configuration Scripts for AutoGen

This directory contains configuration management scripts for the AutoGen project.

## Purpose

These scripts help manage configuration files and settings for the development environment, including:

- VS Code configuration generation
- Development tool settings
- Repository configuration templates
- Environment-specific configurations

## Scripts

- Configuration file generators
- Template processors
- Settings validation scripts
- Configuration migration utilities

## Usage

These scripts are used by the main orchestration scripts in the parent directory. Refer to the main `.scripts/automation/github/README.md` for usage information.

## Dependencies

- PowerShell 7.0+
- Core module functions
