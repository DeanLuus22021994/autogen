# Validation Scripts for AutoGen

This directory contains validation scripts for the AutoGen development environment.

## Purpose

These scripts help validate that the development environment is properly configured, including:

- Checking required environment variables
- Validating development tool installations
- Ensuring proper repository configuration
- Verifying security settings

## Scripts

- Environment variable validation
- Configuration file validation
- Installation checks for required tools
- Repository structure validation

## Usage

These scripts are used by the main orchestration scripts in the parent directory. Refer to the main `.scripts/automation/github/README.md` for usage information.

## Dependencies

- PowerShell 7.0+
- Core module functions
