# Security Scripts for AutoGen

This directory contains security-focused automation scripts for the AutoGen project.

## Purpose

These scripts help identify and remediate security issues in the codebase, including:

- Scanning for exposed tokens and secrets
- Validating repository configuration for security best practices
- Managing secure environment variables
- Preventing accidental commits of sensitive information

## Scripts

- Security scanning utilities
- Token detection patterns
- Remediation functions for security issues

## Usage

These scripts are used by the main orchestration scripts in the parent directory. Refer to the main `.scripts/automation/github/README.md` for usage information.

## Dependencies

- PowerShell 7.0+
- Core module functions
