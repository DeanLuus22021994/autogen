# Common Utilities for AutoGen Automation Scripts

This directory contains common utility functions and shared code used across various AutoGen automation scripts.

## Purpose

These utilities provide consistent functionality and styling for the automation scripts, including:

- Console output formatting
- File handling utilities
- Common configuration validation
- Shared data structures

## Usage

The utilities in this directory are imported by the main module files and orchestration scripts. Refer to the main `.scripts/automation/github/README.md` for complete usage information.

## Design Principles

- Keep functions small and focused on a single task
- Use consistent parameter naming and validation
- Document functions with PowerShell help comments
- Follow PowerShell best practices for error handling
